# 视觉 Agent · 飞书多维表格边栏插件

读取当前记录字段，调用 Grsai nano-banana 生图，写回「结果图」与「状态」。

## 推荐表结构（已内置自动匹配）

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| Prompt | 文本 | 生图提示词（读取 + 保存） |
| 参考图1 ~ 参考图5 | 附件 | 最多 5 张，作为 API 参考图输入 |
| 比例 | 单选 | 如 `9:16`、`1:1`、`16:9` |
| 尺寸 | 单选 | 如 `1K`、`2K`、`4K` |
| 模型 | 单选 | 如 `nano-banana-2` |
| 状态 | 单选 | 插件会写入：`生成中` / `成功` / `失败`（请先在单选选项里添加这三项） |
| 结果图 | 附件 | 生图后自动裁剪缩放为 **1440×1440** 写入 |
| 创建人 / 创建时间 | 系统字段 | 插件不读写 |

## 本地开发

```bash
npm install
npm run dev
```

1. 多维表格 → **插件** → **自定义插件** → 填入 `http://localhost:5173`
2. 选中一行后打开插件
3. **设置** 中填写 Grsai API Key
