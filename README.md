# 视觉 Agent · 飞书多维表格边栏插件

读取当前记录字段，调用 Grsai nano-banana 生图，写回「结果图」与「状态」。支持**批量并行生图**与**失败自动重试**。

## 生图操作

| 按钮 | 说明 |
| --- | --- |
| 生成当前行 | 仅为当前查看的行生图（边栏 Prompt 可覆盖表格） |
| 批量生成选中行 | 表格中勾选多行后并行生图（默认最多 6 条并发） |
| 重跑选中失败行 | 在选中行中，仅对状态为「失败」的行再次生图 |

失败时每条任务会自动重试最多 **2 次**（违规/审核/密钥错误不重试）。

## 推荐表结构（已内置自动匹配）

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| Prompt | 文本 | 生图提示词（读取 + 保存） |
| 参考图1 ~ 参考图5 | 附件 | 最多 5 张，作为 API 参考图输入 |
| 比例 | 单选 | 如 `9:16`、`1:1`、`16:9`、`2:3`、`3:4` 等（与 API 支持的比例一致） |
| 尺寸 | 单选 | 如 `1K`、`2K`、`4K` |
| 模型 | 单选 | 如 `nano-banana-2` |
| 状态 | 单选 | 插件会写入：`生成中` / `成功` / `失败`（请先在单选选项里添加这三项） |
| 结果图像素 | 单选 | 如 `1000*1792`、`1440*1440`、`800*800`、`1000*1000`、`750*1000` |
| 结果图 | 附件 | 按「结果图像素」宽×高裁剪缩放后写入（未填时默认 **1440×1440**） |
| 创建人 / 创建时间 | 系统字段 | 插件不读写 |

## 本地开发

```bash
npm install
npm run dev
```

1. 多维表格 → **插件** → **自定义插件** → 填入 `http://localhost:5173`
2. 选中一行后打开插件
3. **设置** 中填写 Grsai API Key
